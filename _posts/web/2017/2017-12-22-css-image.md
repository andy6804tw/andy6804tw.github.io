圖片不限於容器設定的大小，圖片有多大就多大還有可能超出指定範圍，當容器設定寬為 300px 圖片的寬設定 100% 會與父容器等比例縮放，此外為了適應各瀏覽器高設定 auto 避免圖片因縮放而失真。

```html
<div class="card">
  <img src="#">
</div>
```

```css
img{
  display: block;
  width: 100%;
  height: auto;
}
.card {
  width: 300px;
  margin: 0 auto;
}
```
