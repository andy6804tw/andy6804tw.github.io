## 本文你將會學到
- 使用 git 將專案引入到 VM 虛擬機中
- 在 VM 上測試執行API

## 測試

```
cd RESTful_API_start_kit
git checkout master
npm install
cp .example.env .env
vim .env
```
```
webpack
npm run start
```
