# Intro

## What is Kotlin

- Programming Language developed by JetBrains
- 100% Interoperable with Java
- Concise

## Why Kotlin

- Free
- Concise
- No runtime overhead
- Lots of modern features

## Platforms

- JVM
- Android
- Browser
- Native
