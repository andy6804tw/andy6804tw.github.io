# 控制流程

## if else
`if else` 條件判斷在 Kotlin 中，跟其他語言差不多，讓我們看一下範例。其中顯示範圍例如 4~7 可以使用 `4..7` 表示，等同於 `tot>3 && tot<8`。之後的迴圈使用方式也類似。

```kt
val tot = 5
if(tot<=3){
    println("Low")
}else if(tot in 4..7){ // equivalent of tot>3 && tot<8
    println("Medium")
}else{
    println("High")
}
````

我們也可以直接在變數使用條件判斷，將結果直接賦予數值。

```kt
val num1 = 10
val num2 = 12
val max = if (num1 > num2) num1 else num2
```
